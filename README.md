## Project Overview

This project analyzes the open-source software Python.  
The objective is to understand the philosophy of open source, explore how Python works within a Linux environment, and demonstrate basic Linux shell scripting skills.

Python is one of the most widely used programming languages in the world. It is open source, easy to learn, and supported by a large community of developers.

---

## Repository Contents

This repository contains five shell scripts created as part of the project.

| Script Name | Description |
|-------------|-------------|
| system_report.sh | Displays system information like kernel version, user, uptime, and Linux distribution. |
| package_inspector.sh | Checks whether Python is installed on the system and shows its version. |
| disk_audit.sh | Audits important directories and displays their permissions and disk usage. |
| log_analyzer.sh | Reads a log file and counts occurrences of a keyword such as "error". |
| manifesto.sh | Generates a personalized open source manifesto based on user input. |

---

## How to Run the Scripts

### 1. System Identity Report

```bash
bash system_report.sh
```

### 2. FOSS Package Inspector

```bash
bash package_inspector.sh
```

### 3. Disk and Permission Auditor

```bash
bash disk_audit.sh
```

### 4. Log File Analyzer

Example:

```bash
bash log_analyzer.sh /var/log/syslog error
```

### 5. Open Source Manifesto Generator

```bash
bash manifesto.sh
```
